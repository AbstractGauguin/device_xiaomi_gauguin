Device configuration for Xiaomi Redmi K30 5G
=========================================

The Xiaomi Redmi K30 5G (codenamed _"picasso"_) is a mid range smartphone from Xiaomi.

It was announced in December 2019. Release date was January 2020.

Memory  | 6/8 GB RAM
Shipped Android Version | 10

Battery | Non-removable Li-Po 4500 mAh battery
